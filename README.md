# @onegen/optional

> [!WARNING]  
> ЁЯЪз Very much work-in-progress! ЁЯЪз

`Optional` utility class in TypeScript, based on C++
[`std::optional`](https://en.cppreference.com/w/cpp/utility/optional).

Made as minimal as I could without any dependencies.
